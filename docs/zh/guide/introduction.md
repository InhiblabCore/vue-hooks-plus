通过该章节，你将了解到如何快速开始使用 VueHooks Plus。

::: danger 最低 Node.js 版本
仓库开发、包构建、CI 和文档构建要求使用 **Node.js >= 22.18.0**。VueHooks Plus 在浏览器中的应用运行时使用不受影响。
:::

## ✨ 特性

- 🏄🏼‍♂️ 易学易用
- 🔋 支持 SSR
- 🛸 丰富的 Hooks
- 🏟️ 覆盖大部分业务场景
- 🦾 首选 useRequest，强大的请求中间层
- 🎪 交互式 demo，身临其境
- 🎯 使用 TypeScript 构建，提供完整的类型定义文件
- 🪄 支持按需加载，减少打包体积
- 🤺 演练场，大有用武之地
- 🔐 测试完善，安全可靠
